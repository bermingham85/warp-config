# COMMIT FORMAT

## Trigger
Any git commit.

## Format
```
<what changed - one line>

<why it changed - one line, optional>

Co-Authored-By: Warp <agent@warp.dev>
```

## Examples
Good:
```
Add user authentication endpoint

Needed for mobile app login flow

Co-Authored-By: Warp <agent@warp.dev>
```

Bad:
```
fix stuff
```
