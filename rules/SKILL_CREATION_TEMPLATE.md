# SKILL CREATION TEMPLATE

## Trigger
Need to create a reusable skill.

## Template
```markdown
# {Skill Name}

## When to Use
{One sentence trigger condition}

## Parameters
- param1: {description}

## Steps
1. {concrete action}
2. {concrete action}

## Command
```powershell
{The actual command or script}
```

## Output
{What user gets when done}
```

## Save To
`C:\Users\bermi\.warp\skills\{verb}-{noun}.md`

## After Creating
Add to Memory MCP with entity type "Skill"
